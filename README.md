# Booking site mockup

This is a code bundle for Booking site mockup. The original project is available at https://www.figma.com/design/Z5Z0nNkckxDttfzMQYaLxA/Booking-site-mockup.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

Run `npm test` to execute unit tests once.

Run `npm run test:watch` to run tests in watch mode.
